Paybox payment gate for jigoshop

== Description ==

Russian payment gate Paybox (http://www.paybox.money/) now integrated to JigoShop. Paybox include more than 20 payment system and have only one API and commerce offer. With this plugin integration will more easy!!!

== Installation ==

1. Upload plugin into wp-content/plugins
2. Activate and install the plugin Paybox through the 'Plugins' menu in WordPress
3. Register on http://www.paybox.money/
4. Get and paste settings through the 'Jigoshop->Settings->Payment'
